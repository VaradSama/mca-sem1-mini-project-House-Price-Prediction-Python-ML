House Price Prediction - Python ML
About
A Machine Learning based web application that predicts house prices based on various input features. Built as a MCA Sem 1 Mini Project using Python and Scikit-learn.
Features

Predict house prices based on location, size, rooms, and other features
Data preprocessing and feature engineering
Trained ML model for accurate predictions
Simple and clean web interface

Tech Stack

Language: Python
ML Library: Scikit-learn
Web Framework: Flask
Frontend: HTML, CSS
Dataset: House Price Dataset

How It Works

User enters house details (location, size, rooms, etc.)
Input is passed to the trained ML model
Model predicts the estimated house price
Result is displayed on the web interface
